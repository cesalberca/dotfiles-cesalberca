**Case-study post (AstroKube).**
"AI is not making code more valuable. Is making it cheaper.

This means, that the architecture matters more than ever. AI agents amplify what is already in your codebase, for better or for worse.

That exactly was the nature of the collaboration with AstroKube.

This Spanish company, led by Nicolas Quiceno Benavides and Pasquale Toscano is dedicated to automating cloud deployments. They are top of the line experts in Kubernetes and infrastructure, helping teams to automate deployment processes.

Their current challenge was establishing a solid Frontend foundation for the company from where to build on top of, reusing code and allowing them to go faster while retaining confidence in their Frontend Architecture.

So we built that foundation.

The results?

Errors reduced. Costs lowered. AI integration vastly improved, which actually helped AstroKube land a client thanks to the quality of generated code and development speed, without losing trust in the codebase.

AI has driven code to no longer be a commodity, making other areas like architecture much more critical. That's where the optimization lives. That's where the ROI is now. And that's where you want a human validated architecture.

If your AI agents are generating code that doesn't scale, the agents aren't the problem.

Your architecture is."

**Experience post (Claude Design microinteraction).**
"I recently used Claude Design to tweak a microinteraction on my site and I want to share with you my experience.

I was pleased with the result, as sometimes complex animations can be tricky for AI to really make them satisfying and elegant.

Using already existing code to guide the AI agent was key to the success of this microinteraction.

I showed it code from React Bits of similar animations that inspired me.

At some point, once the prototype was how I wanted I downloaded the HTML code and imported into my blogsite, refactoring it and adapting to the context.

Then I added i18n and a11y, finishing with some tests once I was sure about the interactivity.

It took me around 4 hours of fiddling to make it. Was it worth it? Well, you tell me!
